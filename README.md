psn.proxy
=========
psn.proxy 是利用迅雷离线加速PSN下载的工具
离线迅雷部分代码基于binux的lixian.xunlei项目<https://github.com/binux/lixian.xunlei>
下载将调用aria2c，请保证aria2c在命令行下可用
